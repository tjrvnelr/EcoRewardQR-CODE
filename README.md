# Features EcoReward

- **QR Code Generation**: Generate unique QR codes with specific point values and campaigns via CLI or Web interface.
- **Point Accumulation**: Users scan QR codes to instantly earn points.
- **Rewards Catalogue**: A built-in catalogue of local rewards (restaurants, shops, attractions).
- **Point Redemption**: Interactive web interface for users to redeem their accumulated points for rewards.
- **Data Persistence**: Simple JSON-based storage for users, codes, and transaction history.
- **Dynamic QR Serving**: Built-in route to serve generated QR code images.
